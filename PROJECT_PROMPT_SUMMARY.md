# AX2 프로젝트 요약

## 프로젝트 개요
**AX2**는 실시간 강의 번역 및 자막 생성 웹 애플리케이션입니다. 사용자가 영상 파일을 업로드하면 AI가 자동으로 음성을 인식하고, 설정한 언어로 번역하여 자막을 생성합니다.

## 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **디자인**: Glass morphism 스타일
- **저장소**: LocalStorage (클라이언트 사이드)

## 주요 페이지 (5개)

### 1. 로그인/회원가입 페이지
- 좌측: 브랜드 소개 (보라색 그라데이션)
- 우측: 로그인/회원가입 폼
- 소셜 로그인: Google, KakaoTalk, Naver (이미지 아이콘)
- 언어 전환: 한국어/영어 (기본: 한국어)

### 2. 홈 페이지 (`home.html`)
- **드롭존**: 영상 파일 드래그 앤 드롭 업로드
  - 파일 타입 검증 (MP4, MOV, AVI 등)
  - 파일 크기 제한: 2GB
  - 드래그 시 시각적 피드백
- **번역 설정 모달**: 
  - 원본 언어 선택
  - 번역 언어 다중 선택 (태그 표시)
  - 화자 수, 옵션 설정
  - 번역 후 마이페이지에 자동 저장

### 3. 마이페이지 (`mypage.html`)
- 작업한 강의 목록 (최신순)
- 각 강의 카드 클릭 시 편집 페이지로 이동

### 4. 편집 페이지 (`edit.html`)
- 비디오 플레이어 (재생, 자막 표시)
- 언어 탭 전환 (한국어/영어)
- 전사본 편집 및 제목 수정
- 변경사항 로컬 스토리지 저장

## 핵심 기능

### ✅ 드래그 앤 드롭
- 영상 파일만 업로드 가능
- MIME 타입 및 크기 검증
- 드래그 오버 시 시각적 피드백

### ✅ 번역 프로세스 (시뮬레이션)
1. 영상 스캔 → 2. 음성 인식 → 3. 번역 → 4. 자막 생성 → 5. 완료

### ✅ 자막 표시
- 비디오 재생 시 시간에 맞춰 자막 표시
- 언어별 자막 전환

### ✅ 다중 언어 선택
- 체크박스 기반 UI
- 선택한 언어 태그 표시 및 제거

### ✅ 로컬 스토리지 관리
- 비디오 정보, 자막 데이터 저장
- 플랜 정보 (남은 시간: 100분)

## 디자인 특징
- **테마**: 밝은 테마 (흰색 배경)
- **효과**: Glass morphism, 부드러운 애니메이션
- **반응형**: 모바일/데스크톱 지원

## 사용자 플로우
1. **로그인/회원가입** → 언어 선택 → 소셜 로그인 또는 이메일
2. **홈** → 영상 드래그 앤 드롭 → 번역 설정 → 번역 실행
3. **마이페이지** → 강의 목록 확인 → 강의 클릭
4. **편집** → 자막 확인/편집 → 변경사항 저장
