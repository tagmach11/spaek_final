# AX2 실시간 강의 번역 웹사이트 - 전체 프로젝트 프롬프트

## 프로젝트 개요
AX2는 실시간 강의 번역 및 자막 생성 웹 애플리케이션입니다. 사용자가 영상 파일을 업로드하면 AI가 자동으로 음성을 인식하고, 설정한 언어로 번역하여 자막을 생성합니다.

## 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **스타일링**: 커스텀 CSS (Glass morphism 디자인)
- **아이콘**: Font Awesome 6.4.0
- **데이터 저장**: LocalStorage (클라이언트 사이드)
- **API**: Web Speech API (STT 시뮬레이션)

## 페이지 구조

### 1. 로그인 페이지 (`login.html`)
**기능:**
- 좌측: 브랜드 소개 섹션 (보라색 그라데이션 배경)
  - 로고 버튼 (좌측 상단, mypage.html로 이동)
  - 메인 헤드라인: "만들고, 번역하고, 소통하세요"
  - 서브 헤드라인 및 기능 리스트
- 우측: 로그인 폼
  - 이메일/비밀번호 입력 필드
  - 소셜 로그인 버튼 (Google, KakaoTalk, Naver) - 이미지 아이콘 사용
  - 언어 선택 드롭다운 (한국어/영어) - 기본값: 한국어
  - 모든 텍스트가 선택한 언어에 따라 동적으로 변경
  - 입력 유효성 검사 및 로딩 상태 표시

**스타일:**
- 밝은 테마
- 반응형 디자인
- 애니메이션 효과 (페이드인, 슬라이드)
- 외부 CSS/JS 파일 사용 (`login.css`, `login.js`)

### 2. 회원가입 페이지 (`signup.html`)
**기능:**
- 로그인 페이지와 동일한 레이아웃 및 스타일
- 이름, 이메일, 비밀번호, 비밀번호 확인 입력 필드
- 언어 전환 기능 (한국어/영어)
- 소셜 로그인 버튼 (Google, KakaoTalk, Naver)
- 로고 버튼 (좌측 상단, mypage.html로 이동)

**스타일:**
- 로그인 페이지와 동일한 디자인
- 외부 CSS/JS 파일 사용 (`signup.css`, `signup.js`)

### 3. 홈 페이지 (`home.html`)
**기능:**
- **상단 네비게이션 바:**
  - 좌측: AX2 로고 (mypage.html로 이동)
  - 우측: "Free 0분 남음" 플랜 정보, 로그인 버튼

- **왼쪽 사이드바:**
  - 홈 메뉴
  - 마이페이지 메뉴
  - 각 메뉴 아이템 클릭 시 활성화 표시

- **메인 콘텐츠 영역:**
  - **드롭존 (Drop Zone):**
    - 가로로 긴 박스 형태 (중앙 정렬)
    - 높이: 180px
    - 플랫폼 로고 (YouTube, TikTok, Google Drive)
    - 링크 입력 필드
    - 업로드 버튼
    - 파일 정보 텍스트 ("MP4, MOV, AVI 최대 2GB 지원")
    - **드래그 앤 드롭 기능:**
      - 영상 파일만 업로드 가능 (MIME 타입 검증)
      - 파일 크기 제한: 2GB
      - 드래그 오버 시 시각적 피드백 (border 색상 변경, scale 효과)
      - 클릭 시 파일 선택 다이얼로그 열기
      - 영상 파일 드롭 시 번역 설정 모달 팝업 표시

  - **비디오 미리보기:**
    - 업로드 후 비디오 플레이어 표시
    - 언어 배지 오버레이
    - 자막 오버레이 영역

- **번역 설정 모달 팝업:**
  - 영상 파일 업로드/드롭 시 자동으로 표시
  - 백드롭 클릭 또는 닫기 버튼으로 닫기
  - **설정 항목:**
    - 원본 언어 선택 (드롭다운)
    - 번역 언어 선택 (다중 선택 가능, 체크박스 UI)
      - 선택한 언어는 하단에 태그로 표시
    - 화자 수 선택 (드롭다운)
    - 옵션 (자막 생성, 실시간 번역 체크박스)
  - **번역하기 버튼:**
    - 클릭 시 AI가 영상 스캔
    - 설정한 언어로 자막 생성
    - 번역된 비디오를 마이페이지에 자동 저장

**스타일:**
- 밝은 테마 (흰색 배경)
- Glass morphism 효과
- 애니메이션 효과 (페이드인, 슬라이드)
- 외부 CSS/JS 파일 사용 (`styles.css`, `script.js`)

### 4. 마이페이지 (`mypage.html`)
**기능:**
- **상단 네비게이션 바:**
  - home.html과 동일한 구조
  - 로고 (home.html로 이동)
  - 플랜 정보 및 로그인 버튼

- **왼쪽 사이드바:**
  - 홈 메뉴
  - 마이페이지 메뉴

- **메인 콘텐츠:**
  - 작업한 강의 목록 (최신순 정렬)
  - 각 강의 카드:
    - 썸네일
    - 제목 (편집 가능)
    - 날짜/시간
    - 클릭 시 `edit.html`로 이동 (URL 파라미터로 비디오 ID 전달)

**스타일:**
- home.html과 동일한 디자인 시스템
- 외부 CSS/JS 파일 사용 (`mypage.css`, `mypage.js`)

### 5. 편집 페이지 (`edit.html`)
**기능:**
- **헤더:**
  - 로고 (home.html로 이동)
  - "마이페이지로 돌아가기" 링크
  - 플랜 정보 및 로그인 버튼

- **좌측 편집 영역:**
  - 비디오 제목 (편집 가능, 연필 아이콘 클릭 시 모달)
  - 비디오 플레이어:
    - 재생/일시정지
    - 진행 바
    - 시간 표시
    - 볼륨 조절
    - 재생 속도 조절
    - 전체화면
  - 자막 표시 영역 (비디오 하단 오버레이)
  - 언어 탭 (한국어/영어 전사본 전환)

- **우측 전사본 영역:**
  - 선택한 언어에 따른 전사본 텍스트 표시
  - 편집 가능한 텍스트 영역
  - "변경사항 적용" 버튼 (로컬 스토리지에 저장)

**스타일:**
- 비디오 컨트롤 및 탭 바: 흰색 배경
- 외부 CSS/JS 파일 사용 (`edit.css`, `edit.js`)

## 주요 기능 상세

### 1. 드래그 앤 드롭 파일 업로드
- **파일 타입 검증:**
  - MIME 타입: `video/*`
  - 지원 형식: MP4, MOV, AVI, WEBM, MPEG, MKV, FLV, WMV
- **파일 크기 제한:** 2GB
- **시각적 피드백:**
  - 드래그 오버 시 `drag-over` 클래스 추가
  - border 색상 변경, scale 효과, pulse 애니메이션
- **이벤트 처리:**
  - `dragover`, `dragleave`, `drop`, `dragend` 이벤트
  - `dragCounter`로 중첩 요소 처리

### 2. 번역 설정 모달
- **모달 구조:**
  - 백드롭 (반투명 흰색 배경)
  - 모달 콘텐츠 래퍼 (중앙 정렬)
  - 번역 설정 패널 (흰색 배경, 둥근 모서리)
- **다중 언어 선택:**
  - 체크박스 기반 UI
  - 선택한 언어는 태그로 표시
  - 태그 제거 기능 (X 버튼)
- **모달 닫기:**
  - 닫기 버튼 클릭
  - 백드롭 클릭
  - 페이드아웃 애니메이션

### 3. 번역 프로세스 (시뮬레이션)
- **단계:**
  1. 영상 스캔 중...
  2. 음성 인식 중...
  3. 번역 중...
  4. 자막 생성 중...
  5. 완료
- **결과:**
  - 번역된 자막 데이터 생성
  - 비디오 정보를 로컬 스토리지에 저장
  - 마이페이지에서 확인 가능

### 4. 자막 표시
- **비디오 플레이어 통합:**
  - `currentTime` 이벤트 리스너
  - 시간에 맞는 자막 세그먼트 표시
  - 자막이 없을 때 페이드아웃 효과
- **스타일:**
  - 비디오 하단 중앙 정렬
  - 반투명 배경
  - 폰트 크기 조절 가능

### 5. 언어 전환 기능
- **로그인/회원가입 페이지:**
  - 드롭다운으로 언어 선택 (한국어/영어)
  - 모든 텍스트 동적 변경
  - `data-i18n` 속성 사용
- **기본 언어:** 한국어

### 6. 로컬 스토리지 관리
- **저장 데이터:**
  - 비디오 정보 (제목, 날짜, URL, 자막 데이터)
  - 사용자 설정
  - 플랜 정보 (남은 시간: 100분)
- **데이터 구조:**
  ```javascript
  {
    videos: [
      {
        id: "unique-id",
        title: "비디오 제목",
        date: "2024-01-01 12:00:00",
        videoUrl: "blob:...",
        subtitles: {
          ko: [...],
          en: [...]
        }
      }
    ],
    remainingMinutes: 100
  }
  ```
