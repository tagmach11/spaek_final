첫 번째 자막 만들기 가이드

AX2를 사용하여 영상 자막을 만드는 방법을 단계별로 알아보세요. 초보자도 쉽게 따라할 수 있습니다.

영상에 자막을 추가하고 싶으신가요? AX2를 사용하면 복잡한 과정 없이 몇 분 만에 전문적인 자막을 만들 수 있습니다. 이 가이드에서는 처음 사용하시는 분들도 쉽게 따라할 수 있도록 단계별로 설명드리겠습니다.

1단계: 영상 업로드

먼저 AX2 대시보드에 접속하여 "새 프로젝트" 버튼을 클릭하세요.

  1. 1
    프로젝트 이름 입력

    자막을 만들 영상의 제목이나 프로젝트 이름을 입력하세요.

  2. 2
    영상 파일 선택

    드래그 앤 드롭 또는 파일 선택 버튼을 클릭하여 영상 파일을 업로드하세요. MP4, MOV, AVI 등 다양한 형식을 지원합니다.

  3. 3
    업로드 완료 대기

    업로드가 완료될 때까지 잠시 기다려주세요. 진행 상황이 표시됩니다.

2단계: 언어 선택

영상의 원본 언어와 번역할 언어를 선택하세요. AX2는 30개 이상의 언어를 지원합니다.

원본 언어

영상에서 사용된 언어를 선택하세요. 예: 한국어, 영어, 일본어 등

번역 언어

자막으로 표시할 언어를 선택하세요. 여러 언어를 동시에 선택할 수 있습니다.

3단계: 설정 조정 (선택사항)

더 나은 결과를 위해 몇 가지 설정을 조정할 수 있습니다:

  • 자막 스타일

    폰트, 크기, 색상, 위치 등을 커스터마이징할 수 있습니다.

  • 타이밍 조정

    자막이 나타나는 시점과 지속 시간을 미세 조정할 수 있습니다.

  • 수동 편집

    자동 생성된 자막을 직접 수정하고 개선할 수 있습니다.

4단계: 자막 생성 및 다운로드

모든 설정이 완료되면 "자막 생성" 버튼을 클릭하세요. AI가 영상을 분석하고 자막을 자동으로 생성합니다.

생성 완료 후

  • 1 자막을 미리보기로 확인하세요
  • 2 필요시 수정 후 저장하세요
  • 3 SRT, VTT 등 원하는 형식으로 다운로드하세요

유용한 팁

명확한 음성 품질

배경 소음이 적고 화자가 명확하게 들리는 영상일수록 더 정확한 자막이 생성됩니다.

전문 용어 사전

특정 분야의 전문 용어가 많다면, 사전에 용어집을 등록하면 번역 정확도가 향상됩니다.
