기능

AX2의 강력한 AI 기반 자막 생성 및 번역 기능을 확인하세요.

주요 기능

AX2는 최신 AI 기술을 활용하여 영상 자막 생성과 다국어 번역을 자동화합니다.

실시간 음성 인식
고정밀 음성 인식 기술로 영상의 음성을 정확하게 텍스트로 변환합니다.
다국어 자동 번역
50개 이상의 언어로 자동 번역하며, 여러 언어를 동시에 생성할 수 있습니다.
자막 자동 생성
AI가 자동으로 자막을 생성하고 영상에 맞게 시간 동기화합니다.
영상 자막
다양한 형식의 영상 파일에서 자동으로 자막을 생성합니다.
자막 생성 기능

업로드한 영상에서 자동으로 자막을 생성하고 편집할 수 있습니다.

  • 자동 음성 인식: 고정밀 AI 음성 인식으로 정확한 자막 생성
  • 시간 동기화: 영상의 시간에 맞춰 자막이 자동으로 동기화
  • 자막 편집: 생성된 자막을 직접 수정하고 편집 가능
  • 다양한 형식 지원: SRT, VTT, ASS 등 다양한 자막 파일 형식으로 내보내기
  • 스타일 커스터마이징: 자막 폰트, 크기, 색상, 위치 등 자유롭게 설정
  • 배치 처리: 여러 영상을 한 번에 처리하여 시간 절약
정확도
평균 99% 이상의 정확도로 자막을 생성하며, 전문 용어와 문맥을 이해하여 자연스러운 자막을 제공합니다.
다국어 번역 기능

한 번의 클릭으로 여러 언어로 자막을 번역할 수 있습니다.

  • 50개 이상 언어 지원: 한국어, 영어, 일본어, 중국어, 스페인어, 프랑스어, 독일어 등
  • 동시 다국어 번역: 여러 언어를 한 번에 번역하여 생성
  • 문맥 이해: AI가 문맥을 이해하여 자연스러운 번역 제공
  • 전문 용어 처리: 교육, 비즈니스, 기술 등 분야별 전문 용어 정확히 번역
  • 번역 품질 조정: 번역 스타일과 톤을 조정하여 원하는 번역 생성
  • 용어집 기능: 특정 용어의 번역을 미리 정의하여 일관성 유지
한국어 영어 일본어 중국어 스페인어 프랑스어 독일어 + 43개 언어
파일 관리 및 저장

생성한 자막과 번역 파일을 체계적으로 관리하고 저장할 수 있습니다.

  • 자동 저장: 생성된 자막과 번역 파일을 자동으로 저장
  • 작업 목록 관리: 모든 작업을 한 곳에서 확인하고 관리
  • 검색 기능: 제목, 날짜, 언어 등으로 작업 검색
  • 다운로드: 다양한 형식으로 자막 파일 다운로드
  • 공유 기능: 링크를 통해 자막 파일을 다른 사람과 공유
  • 자동 삭제: 처리 완료 후 2시간 뒤 자동 삭제로 개인정보 보호
자막 편집 기능

생성된 자막을 직접 편집하고 수정하여 완벽한 자막을 만들 수 있습니다.

  • 텍스트 편집: 자막 텍스트를 직접 수정하고 편집
  • 시간 조정: 자막의 시작 및 종료 시간을 미세 조정
  • 자막 분할/병합: 자막을 분할하거나 여러 개를 병합
  • 스타일 설정: 폰트, 크기, 색상, 배경, 위치 등 자유롭게 설정
  • 미리보기: 편집한 자막을 영상과 함께 실시간 미리보기
  • 버전 관리: 편집 이력을 저장하고 이전 버전으로 되돌리기

API 및 통합

다른 서비스와 통합하여 사용할 수 있는 API를 제공합니다.

  • RESTful API: 표준 REST API로 자막 생성 및 번역 기능 사용
  • 웹훅 지원: 작업 완료 시 자동으로 알림 받기
  • SDK 제공: Python, JavaScript 등 다양한 언어용 SDK
  • 플랫폼 통합: YouTube, Vimeo 등 주요 플랫폼과 통합
  • 자동화 워크플로: 반복 작업을 자동화하여 효율성 향상
