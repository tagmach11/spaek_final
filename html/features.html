기능

AX2의 강력한 AI 기반 실시간 언어 번역 기능을 확인하세요.

주요 기능

AX2는 최신 AI 기술을 활용하여 실시간 음성 인식과 다국어 번역을 제공합니다.

실시간 음성 인식
고정밀 음성 인식 기술로 마이크 입력을 실시간으로 정확하게 텍스트로 변환합니다.
실시간 다국어 번역
100개 이상의 언어로 실시간 번역하며, 최대 3개 언어를 동시에 번역할 수 있습니다.
즉시 번역 결과 표시
음성 인식과 동시에 번역 결과가 실시간으로 화면에 표시됩니다.
번역 세션 저장
번역 세션을 저장하고 나중에 다시 확인하거나 다운로드할 수 있습니다.
실시간 번역 기능

마이크 입력을 통해 실시간으로 음성을 인식하고 즉시 번역합니다.

  • 실시간 음성 인식: 고정밀 AI 음성 인식으로 음성을 즉시 텍스트로 변환
  • 즉시 번역: 인식된 텍스트를 선택한 언어로 실시간 번역
  • 다중 언어 동시 번역: 최대 3개 언어를 동시에 번역하여 표시
  • 언어별 패널: 원문과 각 번역 언어를 별도 패널로 구분하여 표시
  • 세션 관리: 번역 세션을 시작하고 종료하여 관리
  • 크레딧 기반: 사용 시간과 번역 언어 수에 따라 크레딧 차감
정확도
평균 99% 이상의 정확도로 음성을 인식하고 번역하며, 전문 용어와 문맥을 이해하여 자연스러운 번역을 제공합니다.
다국어 번역 기능

실시간으로 최대 3개 언어까지 동시에 번역할 수 있습니다.

  • 100개 이상 언어 지원: 한국어, 영어, 일본어, 중국어, 스페인어, 프랑스어, 독일어 등
  • 동시 다국어 번역: 최대 3개 언어를 한 번에 실시간 번역
  • 문맥 이해: AI가 문맥을 이해하여 자연스러운 번역 제공
  • 전문 용어 처리: 교육, 비즈니스, 기술 등 분야별 전문 용어 정확히 번역
  • 자동 언어 감지: 원본 언어를 자동으로 감지하여 설정
  • 실시간 결과 표시: 번역 결과가 즉시 화면에 표시되어 확인 가능
한국어 영어 일본어 중국어 스페인어 프랑스어 독일어 + 43개 언어
번역 세션 관리 및 저장

실시간 번역 세션을 체계적으로 관리하고 저장할 수 있습니다.

  • 자동 저장: 번역 세션이 자동으로 저장되어 나중에 확인 가능
  • 세션 목록 관리: 모든 번역 세션을 한 곳에서 확인하고 관리
  • 검색 기능: 날짜, 언어, 키워드 등으로 세션 검색
  • 다운로드: SRT, VTT, ASS 등 다양한 형식으로 번역 결과 다운로드
  • 세션 상세 보기: 저장된 세션의 원문과 번역 결과를 상세히 확인
  • 자동 삭제: 세션 종료 후 2시간 뒤 자동 삭제로 개인정보 보호
번역 결과 확인 및 다운로드

실시간 번역 결과를 확인하고 다양한 형식으로 다운로드할 수 있습니다.

  • 실시간 확인: 번역 진행 중 원문과 번역 결과를 실시간으로 확인
  • 세션 저장: 번역 세션을 저장하여 나중에 다시 확인
  • 다양한 형식 다운로드: SRT, VTT, ASS 등 다양한 자막 형식으로 다운로드
  • 언어별 필터링: 특정 언어의 번역 결과만 확인하거나 다운로드
  • 시간 정보 포함: 번역 결과에 시간 정보를 포함하여 자막 파일로 활용
  • 세션 상세 보기: 저장된 세션의 전체 번역 내용을 상세히 확인
고급 기능

전문가를 위한 고급 기능들을 제공합니다.

시간 기반 크레딧
사용 시간과 번역 언어 수에 따라 크레딧 차감
언어 설정
원본 언어와 번역 언어를 쉽게 선택하고 변경
세션 제어
언제든지 번역 세션을 시작하거나 종료 가능
사용량 통계
번역 세션 통계와 크레딧 사용량을 한눈에 확인
API 및 통합

다른 서비스와 통합하여 사용할 수 있는 API를 제공합니다.

  • WebSocket API: 실시간 번역을 위한 WebSocket API 제공
  • RESTful API: 번역 세션 관리 및 결과 조회를 위한 REST API
  • SDK 제공: Python, JavaScript 등 다양한 언어용 SDK
  • 실시간 스트리밍: 음성 스트리밍과 번역 결과 수신 지원
  • 자동화 워크플로: 반복 작업을 자동화하여 효율성 향상
