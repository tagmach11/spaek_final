도구

AX2 실시간 언어 번역 서비스를 위한 다양한 도구와 유틸리티를 확인하세요.

API 및 SDK

AX2 실시간 번역 기능을 다른 애플리케이션에 통합할 수 있는 API와 SDK를 제공합니다.

WebSocket API
실시간 번역을 위한 WebSocket API로 음성 스트리밍과 번역 결과를 실시간으로 수신할 수 있습니다.
RESTful API
번역 세션 관리, 결과 조회, 크레딧 확인 등 다양한 기능을 REST API로 제공합니다.
SDK
Python, JavaScript, Java 등 다양한 언어용 SDK를 제공하여 쉽게 통합할 수 있습니다.
통합 가이드
API 문서와 통합 예제 코드를 제공하여 빠르게 시작할 수 있습니다. 개발자 포털에서 상세한 가이드를 확인하세요.
개발자 도구

개발자를 위한 다양한 도구와 유틸리티를 제공합니다.

  • API 테스트 도구: 브라우저에서 직접 API를 테스트할 수 있는 인터랙티브 도구
  • 웹훅 설정: 번역 세션 완료 시 자동으로 알림을 받을 수 있는 웹훅 설정
  • 인증 토큰 관리: API 키 생성 및 관리 도구
  • 사용량 모니터링: API 사용량과 크레딧 소비를 실시간으로 모니터링
  • 에러 로그: API 호출 오류 및 디버깅 정보 확인
  • 코드 예제: 다양한 언어와 프레임워크별 통합 예제 코드
통합 도구

다양한 플랫폼과 서비스에 쉽게 통합할 수 있는 도구를 제공합니다.

브라우저 확장
Chrome, Firefox, Edge용 브라우저 확장 프로그램으로 웹사이트에서 바로 실시간 번역을 사용할 수 있습니다.
모바일 SDK
iOS와 Android용 네이티브 SDK를 제공하여 모바일 앱에 실시간 번역 기능을 통합할 수 있습니다.
협업 도구 통합
Slack, Microsoft Teams, Zoom 등 협업 도구와 통합하여 회의 중 실시간 번역을 제공합니다.
유틸리티 도구

실시간 번역 서비스를 더욱 편리하게 사용할 수 있는 유틸리티 도구입니다.

  • 번역 세션 변환기: 저장된 번역 세션을 SRT, VTT, ASS 등 다양한 자막 형식으로 변환
  • 크레딧 계산기: 예상 사용 시간과 언어 수를 입력하면 필요한 크레딧을 계산
  • 언어 감지 도구: 음성 샘플을 업로드하여 언어를 자동으로 감지
  • 번역 품질 테스트: 번역 정확도를 테스트할 수 있는 도구
  • 배치 처리 도구: 여러 번역 세션을 한 번에 관리하고 처리
  • 백업 및 복구: 번역 세션 데이터를 백업하고 복구하는 도구
지원 및 문서

도구 사용을 위한 상세한 문서와 지원 자료를 제공합니다.

API 문서
