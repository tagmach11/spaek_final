보안

AX2는 사용자의 데이터와 개인정보를 보호하기 위해 최선을 다하고 있습니다.

데이터 보호 정책

AX2는 사용자의 개인정보와 업로드된 영상 데이터를 최우선으로 보호합니다.

자동 데이터 삭제
업로드된 모든 영상 파일과 생성된 자막 데이터는 처리 완료 후 2시간이 지나면 자동으로 완전히 삭제됩니다. 이는 사용자의 개인정보 보호와 데이터 보안을 위한 필수 조치입니다.
  • 영상 파일: 업로드된 원본 영상 파일은 처리 완료 후 2시간 내 자동 삭제
  • 자막 파일: 생성된 자막 파일도 동일하게 2시간 후 자동 삭제
  • 임시 데이터: 처리 과정에서 생성된 모든 임시 파일도 즉시 삭제
  • 복구 불가: 삭제된 데이터는 복구할 수 없으므로 필요한 경우 다운로드하여 보관하세요
데이터 암호화

모든 데이터는 전송 및 저장 중 최고 수준의 암호화를 적용합니다.

암호화 표준
AES-256 암호화를 사용하여 데이터를 보호하며, 모든 통신은 TLS 1.3 프로토콜을 통해 암호화됩니다.
  • 전송 암호화: HTTPS/TLS 1.3을 통한 모든 데이터 전송
  • 저장 암호화: 서버에 저장되는 모든 데이터는 AES-256 암호화
  • 암호화 키 관리: 키는 별도의 보안 저장소에 안전하게 관리
  • 정기 감사: 암호화 시스템은 정기적으로 보안 감사를 받습니다
개인정보 보호

AX2는 사용자의 개인정보를 수집하지 않으며, 최소한의 정보만 요구합니다.

  • 최소 정보 수집: 서비스 이용에 필요한 최소한의 정보만 수집
  • 익명 처리: 업로드된 영상은 개인 식별 정보와 분리하여 처리
  • 제3자 공유 금지: 수집된 정보를 제3자에게 판매하거나 공유하지 않음
  • 쿠키 정책: 필수 쿠키만 사용하며, 추적 목적의 쿠키는 사용하지 않음
  • 개인정보 삭제: 계정 삭제 시 모든 개인정보 즉시 삭제
접근 제어 및 인증

강력한 인증 시스템으로 무단 접근을 방지합니다.

  • 비밀번호 암호화: 모든 비밀번호는 bcrypt 해시 알고리즘으로 암호화 저장
  • 2단계 인증: 선택적 2단계 인증(2FA) 지원
  • 세션 관리: 자동 로그아웃 및 세션 타임아웃 설정
  • IP 주소 로깅: 보안을 위한 접근 IP 주소 기록 (2시간 후 자동 삭제)
  • 비정상 접근 감지: 의심스러운 로그인 시도 자동 감지 및 알림
서버 및 인프라 보안

최신 보안 기술과 모니터링 시스템으로 서버를 보호합니다.

  • 방화벽: 다층 방화벽 시스템으로 무단 접근 차단
  • DDoS 방어: 분산 서비스 거부 공격 자동 차단
  • 정기 업데이트: 보안 패치 및 시스템 업데이트 즉시 적용
  • 침입 탐지: 24/7 실시간 보안 모니터링 및 침입 탐지 시스템
  • 백업 및 복구: 정기적인 데이터 백업 및 재해 복구 계획
  • ISO 27001: 국제 정보보안 관리 표준 준수
보안 인증 및 규정 준수

AX2는 국제 보안 표준과 개인정보 보호 규정을 준수합니다.

GDPR 준수 개인정보보호법 준수 ISO 27001 SOC 2 Type II
  • GDPR: 유럽 일반 데이터 보호 규정 완전 준수
  • 개인정보보호법: 대한민국 개인정보 보호법 준수
  • 정기 감사: 연 1회 외부 보안 감사 실시
  • 보안 취약점 신고: 보안 취약점 발견 시 즉시 신고 및 보상 프로그램 운영
사용자 권한 및 책임

사용자도 보안 유지에 협조해주시기 바랍니다.

  • 강력한 비밀번호: 최소 8자 이상, 영문, 숫자, 특수문자 조합 사용 권장
  • 정기 비밀번호 변경: 3개월마다 비밀번호 변경 권장
  • 공용 네트워크 주의: 공용 Wi-Fi 사용 시 VPN 사용 권장
  • 의심스러운 활동 신고: 이상한 활동 발견 시 즉시 고객센터로 신고
  • 데이터 다운로드: 중요한 자막 파일은 즉시 다운로드하여 보관
중요 안내
업로드된 영상과 생성된 자막은 보관 기간이 만료되면 자동으로 삭제됩니다. 모든 영상은 7일 동안 보관되며, 보관 확장 옵션(Storage Plus/Pro)을 구매하시면 더 긴 기간 보관이 가능합니다. 만료 3일 전부터 알림이 표시됩니다. 필요한 파일은 반드시 다운로드하여 보관하시기 바랍니다. 삭제된 데이터는 복구할 수 없습니다.
