v2.1.0

50+ 언어 지원 확대 및 번역 품질 개선

2025년 1월 15일
NEW

AX2가 이제 50개 이상의 언어를 지원합니다. 더 많은 국가와 지역의 학습자들이 실시간 번역 서비스를 이용할 수 있게 되었으며, 번역 정확도가 크게 향상되었습니다.

주요 변경 사항
  • 50개 이상의 언어 지원 추가 (이전: 30개)
  • 번역 정확도 평균 5% 향상
  • 실시간 지연시간 0.2초로 단축
  • 전문 용어 사전 확대
  • 컨텍스트 인식 번역 기능 개선
v2.0.5

API 통합 기능 강화 및 성능 최적화

2024년 12월 20일

RESTful API 기능이 대폭 개선되었으며, 다양한 플랫폼과의 연동이 더욱 쉬워졌습니다. 또한 전체적인 시스템 성능이 최적화되어 더 빠르고 안정적인 서비스를 제공합니다.

주요 변경 사항
  • RESTful API v2.0 출시
  • Zoom, Teams, LMS 등 주요 플랫폼 연동 개선
  • API 응답 속도 30% 향상
  • 에러 핸들링 및 로깅 시스템 개선
  • API 문서 자동화 도구 추가
v2.0.0

대규모 업데이트: 새로운 UI/UX 및 분석 대시보드

2024년 11월 10일
중요

완전히 새로워진 사용자 인터페이스와 실시간 분석 대시보드를 제공합니다. 번역 품질과 학습 효과를 실시간으로 모니터링하고 최적화할 수 있습니다.

주요 변경 사항
  • 완전히 새로워진 UI/UX 디자인
  • 실시간 분석 대시보드 추가
  • 번역 품질 지표 시각화
  • 반응형 디자인 개선 (모바일, 태블릿, 데스크톱)
  • 다크 모드 지원
  • 접근성 기능 강화
v1.9.0

보안 강화 및 개인정보 보호 기능 개선

2024년 10월 5일

엔드투엔드 암호화가 강화되었으며, 개인정보 보호 기능이 개선되었습니다. GDPR 및 개인정보보호법을 완전히 준수합니다.

주요 변경 사항
  • 엔드투엔드 암호화 강화 (AES-256)
  • 2단계 인증(2FA) 지원
  • 데이터 익명화 기능 개선
  • GDPR 완전 준수
  • 접근 로그 관리 시스템 개선
v1.8.0

자막 커스터마이징 기능 추가

2024년 9월 15일

자막의 위치, 스타일, 색상, 폰트를 자유롭게 커스터마이징할 수 있는 기능이 추가되었습니다. 브랜드 컬러와 로고를 적용하여 일관된 브랜딩을 유지할 수 있습니다.

주요 변경 사항
  • 자막 위치 및 스타일 커스터마이징
  • 브랜드 컬러 및 로고 적용 기능
  • 폰트 크기 및 색상 조정
  • 자막 템플릿 라이브러리 추가
  • 실시간 미리보기 기능
